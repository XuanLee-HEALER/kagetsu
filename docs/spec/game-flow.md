# 游戏流程 / 状态机

## 1. 牌张构成

- 数牌 (suupai):万 (m)、筒 (p)、索 (s),每种 1-9,各 4 张 = 108 张
- 字牌 (jihai):
  - 风牌 (kazehai):东南西北 各 4 张 = 16 张
  - 三元牌 (sangenpai):中發白 各 4 张 = 12 张
- 总计 **136** 张
- 赤宝牌 (aka dora) 可选:0万/0筒/0索 各 1 张,替换原对应一张五饼/五万/五索

## 2. 一局开始

1. 洗牌、码牌成 4 堵,每堵 17 墩 × 2 层。
2. 庄家(亲家)掷骰决定开门(从哪堵开始取牌)。
3. **配牌**:每家轮取 4 张 × 3 轮 + 1 张,亲家 14 张,子家 13 张。
4. 翻**宝牌指示牌**:王牌区从右侧第三张翻开作为 dora 表牌。
5. 王牌区固定 14 张(含已翻 dora 表牌),其中后 4 张为岭上牌。
6. 剩余山牌为活牌山,可摸牌数 = `136 - 14 - 13×4 = 70`。

## 3. 回合循环

```
摸牌 → 判定(暗杠/加杠/自摸) → 弃牌 → 他家判定(吃/碰/明杠/荣和) → 下一家
```

- 摸牌前若手中有暗杠,可宣言;暗杠后翻新 dora,从岭上摸补。
- 摸牌可立即自摸和(若成立)。
- 弃牌后按优先级判定他家行动:
  1. 荣和 (多家可同时和,**头跳**或 **double ron / triple ron** 视规则)
  2. 碰 / 明杠 (同优先级)
  3. 吃 (仅下家)
- 鸣牌后,鸣牌者跳过摸牌,直接进入弃牌阶段(吃碰)或杠后岭上摸。

## 4. 鸣牌(副露)规则

| 类型 | 条件 | 牌的来源 |
|---|---|---|
| 吃 (Chi) | 仅下家弃牌,组成顺子 | 下家 |
| 碰 (Pon) | 任何家弃牌,组成刻子 | 任意 |
| 明杠 (Minkan) | 任何家弃牌,自手三张同 | 任意 |
| 加杠 / 小明杠 (Shouminkan) | 已碰刻子 + 自摸第四张 | 自摸后追加 |
| 暗杠 (Ankan) | 自手四张同,自己回合宣言 | 自摸/手内 |

立直后只能宣言**不改变手牌结构的暗杠**(即拆开后听牌不变、面子不变,且杠的牌不可作雀头)。

## 5. 立直 (Riichi)

### 条件
- 门清(无副露,暗杠不算副露但需保证不破坏听牌结构)。
- 听牌。
- 牌山剩余 ≥ 4 张(确保自己至少能再摸一次)。
- 持有 ≥ 1000 点。

### 宣言
- 出立直棒(1000 点)。
- 切出弃牌横置,标识立直。
- 之后只能摸切(除合法暗杠/和牌)。

### 一发 (Ippatsu)
- 立直后到自己下次摸牌前(一巡内)和牌即一发 +1 番。
- 期间任何人鸣牌(吃碰杠)使一发失效。

### 里宝牌 (Ura Dora)
- 立直和牌时翻开宝牌指示牌下层对应位置作为里 dora。

## 6. 振听 (Furiten)

任意一项成立则**禁止荣和**(自摸不受限):

1. **常态振听**:听牌中任一张已被自己弃过(即使现在不在河中也算)。
2. **同巡振听**:本巡内放过他家可和牌张,直至自己下一次摸牌。
3. **立直振听**:立直后放过任意可和张,直至本局结束(无法解除)。
4. **多面听振听**:多面听中有一张被自己弃过,则其它待牌也禁和。

## 7. 流局 (Ryuukyoku)

### 荒牌流局 (Howaipai)
- 山牌摸尽(每家最多摸 18 张)。
- 计算听牌罚符(见 scoring.md 第 6 节)。
- 亲家听牌则**连庄**,否则下庄。
- 立直棒留下一局。

### 中途流局
| 类型 | 条件 |
|---|---|
| 九种九牌 (Kyuushuu Kyuuhai) | 子家第一摸后,手牌幺九字牌 ≥ 9 种,可宣言流局 |
| 四风连打 (Suufon Renda) | 第一巡四家弃同一张风牌,无人鸣牌 |
| 四杠散了 (Suukaikan) | 第四杠成立瞬间(同一玩家四杠不流) |
| 四家立直 (Suucha Riichi) | 四家全部立直 |
| 三家和了 (Sancha Hou) | 同一弃牌三家荣和(部分规则视为有效,本项目按流局) |

中途流局**不连庄**(部分规则下九种九牌连庄,WRC 不连庄)。

## 8. 局/场/本场

- **东风战**:仅东 1-4 局
- **半庄战**(默认):东 1-4 + 南 1-4
- **西入** (West start):南 4 结束时点数低于 30000 触发(可选规则)
- **场风** (Bakaze):东场所有家场风为东;南场为南。
- **自风** (Jikaze):
  - 东家(起家): 东
  - 南家: 南
  - 西家: 西
  - 北家: 北
- 一局结束后,按规则推进:
  - 亲家和牌或亲家流局听牌:连庄,本场+1
  - 否则:庄移至下家,本场重置(但若下局开始即流局,本场也+1)

## 9. 结束条件

- 半庄战:南 4 结束(若亲家未连庄)。
- **延长**:南 4 亲连庄结束时若全员 < 30000 → 西入,继续打到任一家 ≥ 30000 或西 4 结束。
- **箱下** (击飞):任一家点数 < 0 强制结束(可选规则)。
- 终局后按持点排序,加 Uma/Oka(本项目暂留接口)。

## 10. 状态机概览

```text
GameState
├─ Round { wind, kyoku, honba, riichi_sticks }
├─ Wall { live, dead, dora_indicators[], ura_dora_indicators[] }
├─ Players[4] { hand, melds, river, score, riichi, ippatsu, furiten }
├─ Turn { current_seat, phase: Draw/Discard/Call/Win/Ryuukyoku }
└─ History { turns[], events[] }
```

阶段 (Phase):
1. `Deal` - 配牌
2. `Draw` - 当前家摸牌
3. `Discard` - 当前家弃牌(可宣言暗杠/加杠/自摸/立直)
4. `Call` - 等待他家鸣牌/荣和决策(超时或全部跳过则进入下一巡)
5. `RoundEnd` - 和牌或流局结算
6. `GameEnd` - 整场终局
