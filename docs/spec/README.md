## 默认规则配置

本项目以 **WRC 2022** 为主基,下列细则为默认值,可在 `GameConfig` 中切换:

| 项目 | 默认 | 说明 |
|---|---|---|
| 半庄/东风 | 半庄 (East+South) | |
| 食断 (Kuitan) | 启用 | 鸣牌后断幺九成立 |
| 一发 / 里宝 | 启用 | |
| 赤宝牌 | 启用,各花色五各 1 张 | |
| 古役 (人和、三连刻、大车轮等) | 关闭 | 用户可开启 |
| 多家荣和 | 头跳 (atamahane) | |
| 八连庄 | 关闭 | |
| 西入 | 启用 | 南 4 < 30000 触发 |
| 击飞 (箱下) | 关闭 | 点数为负不强制结束 |
| 累计役满 (Kazoe) | 启用 | 13+ 番按役满计 |
| Uma | ±15 / ±5 | |
| Oka | +20000 (起始 25000,目标 30000,差额给一位) | |

## 实现优先级

1. **正确性 > 完整性 > 性能**:规则细节常见 bug 高发,优先单元测试覆盖。
2. 算法选择遵循 Pike 准则:`[u8; 34]` 表手牌,DP/回溯求解,不引入复杂结构。
3. 边界用例集中放在 `tests/spec_cases.rs`(从天凤/雀魂等已知牌谱抽样)。
